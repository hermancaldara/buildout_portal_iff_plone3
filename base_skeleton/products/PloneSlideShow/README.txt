PloneSlideShow (0.9.0)

 <b><i>PloneSlideShow</i></b> is a tool for presentation of random images 
 and news in Plone sites.

Information

 * Please read the doc/LICENSE file to learn about PloneSlideShow licence of
   use.

 * Read the doc/ChangeLog to find out what has changed since the last version.

 * Read the doc/TODO to find out what will be made in future versions.

 * doc/CREDITS has all PloneSlideShow contributors, thanks to all.

Installation

 * Please read the doc/DEPENDENCIES file to find out what products the PloneSlideShow rely on.

 * Read the doc/INSTALL file read the installation instructions.

Usage

  1. Unpack the tarball.

  2. Copy or link the 'PloneSlideShow' directory into the 'Products'
     directory of your 'INSTANCE_HOME / SOFTWARE_HOME'.

  3. Restart Zope.

  4. Use the 'portal_quickinstaller' in Plone Site to install it. (Easier, jaster and clean!)

    or:

    * Use the link Preferences -> Add / Remove Products in Site Plone ;-)

  5. The 'CMFPublicator Tool' installation is required. (For more information, visit http://www.zope.org/Members/ferri/CMFPublicator/)

  6. Add the line <b> &lt;tal:block metal:use-macro="context/global_slide/macros/panel" /&gt; </b> desired ZPT.
